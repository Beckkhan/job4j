Использование утилиты jconsole для оценки работы
программы заявок из второго модуля.

Heap Memory Usage
Максимальный размер HEAP растет с течением времени до значения 160Мб,
потом включается GC и после этого размер HEAP составляет 4Мб.

Threads
Пиковое количество работающих потоков при запуске - 15.
После чего значение снижается до 13.

CPU Usage
Максимальная нагрузка на процессор при работе программы - 0,5%.
